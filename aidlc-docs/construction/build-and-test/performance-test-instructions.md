# Performance Test Instructions

## Estado: N/A (justificado)

Este proyecto es una app personal para 2 usuarios concurrentes como máximo, en capa gratuita de Supabase/Vercel. No se han definido (ni tiene sentido definir) objetivos de throughput, tiempo de respuesta bajo carga, o usuarios concurrentes más allá de 2 — ver `nfr-requirements.md` de cada unidad y la decisión de desactivar la extensión de Resiliencia en Requirements Analysis.

No se generan pruebas de carga/estrés. Si el proyecto creciera más allá de su alcance personal original, esta sección debería revisarse.
